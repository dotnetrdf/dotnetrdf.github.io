Namespaces

This page serves as a quick reference to our main namespaces, there are many more child namespaces not detailed here.

Namespaces Purpose Documentation
VDS.RDF Core Classes and Interfaces for RDF representation and access User Guide
VDS.RDF.Configuration Classes that can be used to declare application configuration in RDF Configuration API
VDS.RDF.Ontology Classes for accessing Graphs in a more resource-centric manner Ontology API
VDS.RDF.Parsing Classes for Parsing concrete RDF syntaxes Reading RDF
VDS.RDF.Query Classes for Querying RDF Querying with SPARQL
VDS.RDF.Storage Classes for storing RDF in various Triple Stores Triple Store Integration
VDS.RDF.Update Classes for supporting SPARQL Update Updating with SPARQL
VDS.RDF.Web Classes designed specifically for deploying RDF in ASP.Net applications ASP.Net Integration
VDS.RDF.Writing Classes for serializing RDF in concrete syntaxes Writing RDF
